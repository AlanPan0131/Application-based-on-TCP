# 網路程式設計
## Lab 3 – Simple Mail Client

請修改範例，寫一個 Mail Client 程式，這一個程式至少可以達成以下的功能：

(1)   秀出 mailbox 中有幾封 mail；

(2)   刪除第 nn 封信；

(3)   列出信箱中每一封 mail 的寄信時間、信件主題、寄件人、收件人;

(4)   列出第 nn 封信的內容 (如信件已編碼需解碼)。
